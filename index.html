Transparent reasoning model

Deterministic pediatric anemia algorithm

Every branch uses explicit facts and outputs a diagnostic pattern plus confirmatory steps. No generative model participates in inference.

1

Establish scope and urgency

Age, local ranges, instability, bleeding, severe anemia, other cytopenias, blasts, lead level, and microangiopathy flags.

2

Confirm anemia

Hemoglobin is compared with the local lower limit; if absent, the age/sex-specific AAP fallback is used only from 6 months to <18 years.

3

Classify morphology

MCV is categorized as microcytic, normocytic, or macrocytic using local limits first. RDW and transfusion context modify interpretation.

4

Classify marrow response

Reticulocyte response separates loss/destruction from impaired production; the user supplies the laboratory-context interpretation rather than a universal cutoff.

5

Apply morphology-specific rules

Iron/inflammation/globin/lead; bleeding/hemolysis; renal/inflammatory/marrow; or megaloblastic/non-megaloblastic branches.

6

Rank and expose provenance

Rules merge into a ranked pattern differential. The interface shows all matched rule IDs, findings, cautions, next tests, and citations.

Microcytic branch

  1. Ferritin relative to AAP threshold
  2. CRP and inflammatory iron restriction
  3. sTfR/ferritin index if available
  4. Newborn screen, HbA2/globin testing
  5. Lead and exposure history
  6. Rare iron-loading/sideroblastic pathway

Normocytic branch

  1. High retic: blood loss vs hemolysis
  2. Hemolysis labs, DAT, smear
  3. Low retic: renal/inflammation/viral
  4. Other cytopenias and marrow red flags
  5. Chronic hemolytic disease + abrupt reticulocytopenia

Macrocytic branch

  1. Reticulocytosis vs production failure
  2. B12, folate, smear
  3. TSH, liver, medications
  4. Copper and nutrition/malabsorption
  5. Congenital phenotype or progressive cytopenias
