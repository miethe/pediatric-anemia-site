**Dependencies**: None — first phase; P2/P3/P4 all consume its contracts

`schemas/*` + `scripts/validate-kb.mjs` surface), general-purpose executor (build);
task-completion-validator gate; **karen milestone review at P1 exit** (decisions block §2 —
contract sanity before three parallel workstreams build on these contracts)

join point proving P1-T1's design and P1-T2..T5's schema builds agree.

enumerates every human gate with owner=human; `npm run check` green + task-completion-validator.
