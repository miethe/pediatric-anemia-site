SQ-3 proved the current failure mode is actively dangerous. Growth/kidney fail at `src/units.js:167`
(**not** the evidence registry â€” a correction to the seeded context), throw `UnitRejectionError` with

misattributed. The P2-03 eligibility gate makes this unreachable through the UI; the refusal state is
built anyway as **defence in depth**, as a distinct third state alongside success and
input-rejection, and **never** as a reuse of `showInputRejection` (`src/app.js:686-699`).
